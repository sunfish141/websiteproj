Basic JS Datatypes

Strings

Strings are one of the most common datatypes used in Javascript. They are used to store and manipulate text. A string is represented by zero or more characters written inside quotes. In javascript, strings are primitive datatypes and immutable.

Numbers

Numbers are another one of Javascript's most basic datatypes. These include any number, such as whole numbers and decimals. Numbers stored are subject to floating-point imprecision, meaning that their precision will be lost if the number becomes too precise. Unlike many other languages, Javascript does not define different types on numbers such as ints or floats.

Booleans

Booleans are values that can contain only the values 'true' and 'false', with no other possible values. These are often used in conjunction with if statements, where the if statement will return a boolean value to the computer. They are also used when a variable needs only two possible values, and for controlling program flow.

NULL

Null refers to a primitive datatype which has only one value - null. This means that the variable null is assigned to has 'no value'. Null is an object, meaning it must be assigned by the user to a value. An object can be emptied by setting it to null.

Undefined

Undefined refers to the default value which Javascript passes to all values that are not given one at declaration. Undefined represents the lack of a value in a variable. The difference between null and undefined is that null must be set - null represents the intentional lack of a value, and undefined represents are variable that has not yet been set.
